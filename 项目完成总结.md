## 使用说明

### 快速启动
1. 确保已安装 Node.js 16+ 和 npm 7+
2. 在项目目录执行 `npm install`
3. 执行 `npm start` 启动项目
4. 浏览器访问 http://localhost:3000

### 功能使用
1. **浏览主页**：了解平台整体功能
2. **选择实验**：点击实验卡片进入具体实验
3. **调节参数**：使用控制面板实时调节参数
4. **观察现象**：通过3D视图观察物理现象
5. **分析数据**：查看图表和定量分析结果

### 演示建议
1. **主页展示**：展示平台整体架构和特色
2. **实验演示**：重点演示2-3个代表性实验
3. **技术讲解**：说明自主开发的技术创新
4. **教学价值**：强调实际应用价值和推广前景
